# Crack vs Non-Crack Wall Detection using Transfer Learning

## 📌 Overview

This project focuses on detecting wall cracks using deep learning and transfer learning techniques. The model classifies images into two categories:

* Crack
* Non-Crack

The goal is to assist in automated infrastructure inspection by providing a fast and accurate image-based classification system.

---

## 🚀 Features

* Binary image classification (Crack vs Non-Crack)
* Transfer Learning for improved performance
* High accuracy (~99%)
* Lightweight and efficient model
* Easy to integrate into real-world applications

---

## 🧠 Model Details

* Framework: TensorFlow / Keras
* Approach: Transfer Learning
* Pretrained Model: (e.g., MobileNetV2 / ResNet50 — update if needed)
* Input Size: 128x128
* Output: Binary Classification

---

## 📊 Performance

* Accuracy: ~99%
* Loss: Low and stable
* Evaluation: Tested on unseen validation dataset

---

## 📁 Dataset

The dataset consists of labeled images:

* Crack images
* Non-Crack images

Data is preprocessed and resized before training.
